# scala-intro
An introduction into the Scala programming language

## Installation 
1. First download scala: 
```sudo apt-get install scala ```

2. Then create a .Scala file
```vim hello.Scala```

3. Populate the file as follows: 
``` scala
object World 
{ 
    def main(args: Array[String])  
    { 
        println("Hello World!")  
    } 
}

```

4. Run the programme: 
```scala World```
